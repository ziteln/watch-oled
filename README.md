# 基于 STM32 的多功能智能手表设计

## 一、项目简介
本项目设计了一款基于 STM32F103C8T6 单片机的多功能智能手表，具备实时计步、电子时钟、OLED 显示及蓝牙数据传输功能。系统采用模块化设计，无需 PCB 打样，通过面包板与杜邦线搭建硬件电路，适合课程设计快速实现与演示。

## 二、功能概述
- ✅ 实时计步（MPU6050 六轴传感器）
- ✅ 电子时钟（RTC / 定时器实现）
- ✅ OLED 显示（0.96 寸 I2C SSD1306）
- ✅ 蓝牙数据传输（HC-05 经典蓝牙）
- ✅ 低功耗便携设计（3.7V 锂电池供电）

## 三、硬件组成
| 模块名称 | 型号 | 说明 |
|--------|------|------|
| 主控 | STM32F103C8T6 最小系统板 | “蓝色药丸”板 |
| 显示 | 0.96 寸 OLED | I2C 接口，SSD1306 驱动 |
| 传感器 | MPU6050 | 六轴加速度 + 陀螺仪 |
| 蓝牙 | HC-05 | 经典蓝牙串口模块 |
| 电源 | 3.7V 锂电池 + TP4056 | 可充电方案 |
| 稳压 | AMS1117-3.3 | 为 MCU 与外设供电 |
| 交互 | EC11 旋转编码器（可选） | 菜单切换 / 清零 |

### 计步算法简述
采用加速度阈值 + 峰值检测法：
1. 读取 MPU6050 Z 轴加速度
2. 滑动窗口滤波
3. 超过设定阈值判定为一步
4. 防抖处理，避免误判

## 八、注意事项
1. OLED 与 MPU6050 共用 I2C 总线，地址分别为：
   - OLED：0x3C
   - MPU6050：0x68
2. HC-05 仅支持 Android 设备，iOS 不兼容
3. 所有 I2C 信号线需加上拉电阻（4.7kΩ）
4. 锂电池供电时注意正负极，防止短路
